# Runiq.Rag

Runiq.Rag provides retrieval-augmented generation primitives for Runiq.Net applications.

It includes abstractions and default implementations for document chunking, embedding generation, vector storage, retrieval, and search result mapping.

## Installation

```powershell
dotnet add package Runiq.Rag --prerelease
```
